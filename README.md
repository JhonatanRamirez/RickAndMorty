# RickAndMorty
Tecnologia usadas para el desarrollo de esta actividad:
1. Android Nativo Java.
Librerias Usadas:
1. Volley: Biblioteca HTTP que facilita y agiliza el uso de redes en apps para Android(https://developer.android.com/training/volley?hl=es-419#:~:text=Volley%20es%20una%20biblioteca%20HTTP,autom%C3%A1tica%20de%20solicitudes%20de%20red)
2. Lottie: Es un formato de archivo para animación vectorial.
3. Firebase: Se utilizo para la autenticación con Google.
4. Butterknife: Es una librería que nos facilita la tarea de relacionar los elementos de las vistas con el código en nuestras aplicaciones Android (https://javautodidacta.es/butter-knife-en-android)
5. Glide: Es una popular librería Android de código abierto para cargar imágenes, videos y GIFs animados. (https://devexperto.com/glide-android/)

Se consume el API de RICKANDMORTY para efectos del desarrollo.

Se implementa una arquitectura MVP.
